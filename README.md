# whatsapp-lottie-funny

built in 5 minutes with gemini.

### usage
1. cd whatsapp-lottie-funny
2. npm install
3. node index.js
4. select connect to whatsapp and scan qr
5. select import png to build your .was sticker OR select .was and send to pick an existing one
6. navigate through the steps (output folder, filename, animation, metadata)
7. enter the phone number (with country code, e.g. +1234567890) to send the sticker

i do not give a single flying fuck about what people make with it, nor if it gets their accounts banned, nor if it is patched by today. no license.
